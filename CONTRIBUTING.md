# Contributing

This is a gitgrip workspace manifest. Contributions to individual projects should go to their respective repos:

- **Codi**: https://github.com/laynepenney/codi
- **gitgrip**: https://github.com/laynepenney/gitgrip

## Workspace Contributions

For changes to the workspace itself (manifest, envsetup, documentation, skill files):

1. Install [gitgrip](https://github.com/laynepenney/gitgrip)
2. Initialize the workspace: `gr init https://github.com/laynepenney/codi-gripspace.git`
3. Create a branch: `gr branch feat/your-change`
4. Make changes
5. Submit a PR: `gr pr create -t "description" --push`

## Reporting Issues

- Workspace issues: https://github.com/laynepenney/codi-gripspace/issues
- Codi issues: https://github.com/laynepenney/codi/issues
- gitgrip issues: https://github.com/laynepenney/gitgrip/issues
